Module 1 - Working on a Web Application, Variables

Introduction

Welcome to Module 1 on Variables in JavaScript! In this lesson, students will learn about variables, operators, and basic arithmetic in JavaScript. We will explore how to declare variables using 'let' and 'const', perform operations such as exponentiation, modulo, incrementing, and decrementing, and learn to solve problems by applying these concepts. By the end of this module, students will be able to manipulate and calculate values precisely, a foundational skill for any aspiring web developer.

  2. Accessing the JavaScript interpreter: the console
  3. Adding code to a page using VS Code
  4. Using the console to troubleshoot problems
  5. Adding comments to the code
  6. Formatting code automatically
  7. Understanding primitive types
  8. Using arithmetic operators
  9. Joining strings with the + operator
  10. Determining type with the typeof operator

Sprint Challenge Problems

  1. Calculate the num1 raised to the power of num2 and add num3 to the result. Then, add the remainder of 10 divided by 3 to the previous result.

    Procedure:

    1. Calculate num1 to the power of num2 using the exponentiation operator (**).
    2. Add num3 to the result of the exponentiation.
    3. Use the modulo operator (%) to find the remainder of 10 divided by 3.
    4. Add the remainder to the result obtained after adding num3.

    Code:

                  
                  
                  
    let result1;
    
                  
                  
    const num1 = 6;
    
                  
                  
    const num2 = 5;
    
                  
                  
    const num3 = 938460;
    
                  
                  
    result1 = (num1 ** num2) + num3;
    
                  
                  
    result1 += (10 % 3);

  2. Find the average of two ages, increment the result, declare a new variable, decrement it, and finally, reduce the previous result by this new variable's value.

    Procedure:

    1. Declare result2 and calculate the average of age1 and age2.
    2. Increment result2 by 1 using the increment operator (++).
    3. Declare a new variable with a unique name and assign it the value 10.
    4. Decrement the new variable by 1 using the decrement operator (--).
    5. Subtract the value of the new variable from result2.

    Code:

                  
                  
                  
    let result2;
    

                  
    const age1 = 38;
    
                  
                  
    const age2 = 26;
    
                  
                  
    result2 = (age1 + age2) / 2;
    
                  
                  
    result2++;
    
                  
                  
    let tempVar = 10;
    
                  
                  
    tempVar--;
    
                  
                  
    result2 -= tempVar;
